# Contractor Time Tracking SaaS Admin

Standalone admin dashboard for owners and managers.

## Features

- Workspace overview
- Project and client management
- Approval queue
- Reporting
- Billing settings shell

## Local development

1. Copy `.env.example` to `.env.local`.
2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
npm run dev
```
